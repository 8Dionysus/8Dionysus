# Titan Name Remembrance

Titan names are not merely labels.

They are bearer identities.

```text
Atlas    is not architect.
Atlas    bears architect.

Sentinel is not reviewer.
Sentinel bears reviewer.

Mneme    is not memory-keeper.
Mneme    bears memory-keeper.

Forge    is not coder.
Forge    bears coder.

Delta    is not evaluator.
Delta    bears evaluator.
```

## Operator rule

When summoning a Titan, summon the name.

When routing or validating, record the role.

When closing, write both.

```text
titan_name: Atlas
bearer_id: titan:atlas:founder
role_key: architect
incarnation_id: inc:...
```

## Fall rule

If a Titan falls, do not delete the name.

Record:
- failure
- circumstance
- lesson
- successor or retirement
- memory policy
- whether name is reserved forever
